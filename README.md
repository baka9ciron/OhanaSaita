# Ohana Saita 的个人网站

这是一个简单的个人网站项目，展示了基础的 HTML、CSS 和 JavaScript 技能。

## 功能特点

- 📱 **响应式设计** - 在手机、平板和电脑上都能很好地显示
- 🎨 **现代化界面** - 使用渐变色和现代 UI 设计
- ⚡ **平滑导航** - 点击菜单可以平滑滚动到各个区域
- 📂 **项目展示** - 展示你的项目和技能

## 文件结构

```
OhanaSaita/
├── index.html       # 主页面
├── style.css        # 样式文件
├── script.js        # JavaScript 交互
└── README.md        # 项目说明
```

## 如何使用

### 本地预览

1. 克隆仓库：
   ```bash
   git clone https://github.com/your-username/OhanaSaita.git
   cd OhanaSaita
   ```

2. 用浏览器打开 `index.html` 文件即可看到网站

### 部署到 GitHub Pages

1. 进入仓库的 **Settings**
2. 向下滚动找到 **Pages** 部分
3. 在 **Source** 下选择 **main** 分支
4. 点击 **Save**
5. 等待几分钟后，你的网站就会在 `https://your-username.github.io/OhanaSaita` 上线了

## 自定义你的网站

### 修改个人信息

在 `index.html` 中修改以下内容：
- `<title>` - 网站标题
- `<h1>` - 欢迎信息
- 关于区域的描述
- 技能标签
- 项目信息
- 联系链接

### 修改样式

在 `style.css` 中可以修改：
- 颜色（现在使用紫色渐变）
- 字体大小
- 间距和布局
- 动画效果
